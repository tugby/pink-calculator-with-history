# 🌸 Calculator with history feature

A clean, modern calculator with a soft pink aesthetic and a persistent history log.

## ✨ Features
* **History Modal:** Keeps track of all your calculations during the session.
* **Modern UI:** Soft pink color palette with smooth shadows.
* **Responsive Design:** Works perfectly on both desktop and mobile.

## 🛠️ Technologies used
* **HTML5** - Structure
* **CSS3** - Pink styling and flexbox/grid layouts
* **JavaScript** - Logic and history management
